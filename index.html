02 · Security model

Zero-permission by design.

The most common question about a new trading tool is: what can it do to my account? The answer here is nothing — because it never has access in the first place.

✓ What it does

  • Reads public Binance market data — same as binance.com without login
  • Scans Spot + Futures pairs every few seconds
  • Stores your trading journal locally on your machine only
  • Sends anonymous machine ID + subscription status to license server

✗ What it cannot do

  • See your Binance balance or positions
  • Place, modify, or cancel trades on any exchange
  • Withdraw funds — no API keys exist in the system
  • Identify you on Binance — there is no account link

Network activity, in full

Outbound connections only to: api.binance.com fapi.binance.com alternative.me and the license endpoint. No user behaviour telemetry. No trade data leaves your machine.

The v5.5.0 beta is not yet code-signed. Windows SmartScreen will warn on first launch. A Sectigo certificate is planned for v5.6. If you prefer not to run unsigned binaries — wait for the signed release.

04 · Questions you should ask

The honest answers.

Before installing any unknown executable that touches a crypto exchange, you should ask these questions about every tool. Here are the answers for this one.

Does this place trades for me?
No. CryptoSignal PRO is read-only. It displays signals; you place trades yourself on the exchange of your choice.
Do I need a Binance API key?
No. The app uses public Binance endpoints only — the same data visible on binance.com without logging in.
What happens after my 150 free hours?
The app switches to a limited view (top 3 pairs, no filters). You can subscribe to PRO to restore full access, or keep using the limited view for free.
Are the "win rate" numbers real trading results?
No. They are results from 90-day backtests on historical data. Past performance does not predict future results. There is no live, audited trading track record. Treat the scoring as a filtering aid, not a prediction.
Why does Windows warn when I run the installer?
The beta isn't code-signed yet. A Sectigo certificate is planned for v5.6. Until then, SmartScreen will warn — this is normal for unsigned software. Click "More info → Run anyway", or wait for the signed release if you prefer.
Is the source code available?
Currently closed-source. The GUI and indicator layers are planned to be open-sourced in v5.7 while the calibration engine remains proprietary.
Which exchanges are supported?
Binance Spot and Futures in v5. Bybit and OKX are on the roadmap (v5.6 and v5.7).
Can I cancel my subscription anytime?
Yes. Cancel from your account page — access continues until the end of the paid period.
