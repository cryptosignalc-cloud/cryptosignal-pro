### System Requirements
- Windows 10/11 (64-bit)
- 4 GB RAM (8 GB recommended)
- 500 MB disk space
- Stable internet connection

### Download
1. Go to [Releases](https://github.com/cryptosignalc-cloud/cryptosignal-pro/releases/latest)
2. Download `CryptoSignal_PRO_v5.5.0.zip`
3. Extract the archive
